# HU-02 — Referencias visuales

Se documentan referencias visuales para orientar el diseño del proyecto Rent&Go.

## Referencias tomadas

- Sitios de alquiler de vehículos con catálogo visual.
- Interfaces con tarjetas de vehículos.
- Formularios simples para búsqueda por fecha.
- Diseños responsive para mobile y desktop.

## Decisiones visuales

- Usar tarjetas para mostrar vehículos.
- Mostrar imagen, nombre, tipo de vehículo y precio.
- Mantener una navegación clara y simple.
- Priorizar diseño responsive.

## Relación con el proyecto

Estas referencias sirven como base para el diseño de la interfaz de Rent&Go en próximos pasos.
