# 04 - Sprint Review + Mini Retrospective

## Objetivo del paso

Cerrar la práctica con inspección de resultados y mejora continua.

## Tiempo estimado

10-15 minutos.

## Sprint Review (guion breve)

1. Mostrar tablero final y estado de tarjetas.
2. Mostrar el PR mergeado y su issue asociado.
3. Confirmar qué historias quedaron completadas y cuáles no.
4. Validar si se cumplió el objetivo del mini sprint.

## Mini Retrospective (Start / Stop / Continue)

Completar en equipo:

- **Start:** ¿Qué vamos a empezar a hacer en el próximo sprint?
- **Stop:** ¿Qué práctica debemos dejar de hacer?
- **Continue:** ¿Qué funcionó bien y debemos mantener?

## Evidencias para “aprobado”

1. Link al tablero con tarjetas movidas.
2. Link al issue de épica.
3. Links a 2 historias de usuario.
4. Link al PR mergeado.
5. 2 conclusiones de retrospectiva escritas.

## Resultado esperado

- Cierre formal de sprint realizado.
- Acuerdos de mejora definidos por el equipo.

## Si falla, revisar

- Falta evidencia de PR mergeado.
- Historias sin criterios claros.
- No se registraron conclusiones de retrospectiva.
